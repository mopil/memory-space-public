펀더는 하방, 내러티브는 상방 — 시계열에 따른 의사결정 철학

발견 맥락

2026-04-30 사용자가 VELO narrative 트레이딩 분석 중 직접 정립. 케인즈 미인 콘테스트 프레임 (keynesian-beauty-contest-narrative) 위에 올라가는 메타 철학. 메타원칙 0(“자본은 수급+내러티브가 집중하는 곳으로 흐른다”)을 시계열 축에서 재해석한 통합 프레임.

핵심 3개 명제

명제 1: 가격 결정 변수의 분리

  • 펀더멘털 = 하방 결정. 얼마나 하락해도 회복 가능한가의 한도
  • 내러티브 = 상방 결정. 얼마나 상승할 수 있나의 한도
  • 두 변수는 독립적. 한쪽 강함이 다른쪽 강함 보장 안 함
  • VELO 사례: 펀더(GP -16%)가 하방 약함, 내러티브(Raptor 4)가 상방 강함

명제 2: 시계열에 따른 영향력 변화

  • 펀더 → 가치로 전환되는 데 시간 필요 (속도 느림). 시간이 지나야 가격에 반영
  • 내러티브 → 수급으로 즉시 반영 (속도 빠름). 단기 가격 결정
  • 시계열 길어질수록: 펀더 영향력 증가
  • 시계열 짧아질수록: 내러티브 / 수급 / 시장 심리 영향력 증가
  • 같은 종목이라도 시계열에 따라 정반대 결론 가능

명제 3: 시계열 결정이 의사결정의 출발점

  • 투자자가 가늠하는 시계열이 모든 분석을 좌우
  • 시계열 안 정하면 펀더와 내러티브 변수가 충돌해 분석 무력화
  • “시계열을 어느 정도로 가늠할 것인가”가 의사결정의 핵심 변수

사용자 스타일 정의

시계열 짧음 → 내러티브 / 시장 심리 우선이 디폴트 모드

  • 장기투자 비선호 (최대 시계열 6개월)
  • 회전율 높은 스윙 / 어닝플 / 종배 선호
  • → 펀더가 가치로 전환될 시간 없음
  • → 시장 심리 / 군중 매핑 / catalyst 타이밍이 1차 변수
  • → 케인즈 미인 콘테스트 모드가 디폴트 (keynesian-beauty-contest-narrative)

의사결정 프로토콜

진입 전 순서:

1단계: 시계열 가늠 “이 베팅 몇 개월 보유할 것인가?”

  • < 3개월: 단기 모멘텀 (어닝플, 이벤트 드리븐)
  • 3-6개월: 중기 스윙 (테마 사이클)
  • 6-12개월: 균형 (펀더 1차 전환 시점)

  • 12개월: 장기 (펀더 풀 전환)

2단계: 시계열별 변수 우선순위 적용

  • 단기/중기 (사용자 디폴트): 내러티브 / Float / catalyst / 군중 픽업 단계 우선
  • 장기: 펀더 / 마진 / 매출성장률 우선
  • 균형 구간: 두 변수 동시 평가, 어긋나면 진입 보류

3단계: R/R 비대칭 측정

  • 펀더로 하방 측정 (예: -30%)
  • 내러티브로 상방 측정 (예: +120%)
  • R/R 비율 ≥ 2 : 1 이면 진입 적격
  • 사용자 스타일: 비대칭 성사 + 시계열 짧음 → 내러티브 의사결정

동일 종목, 다른 시계열, 정반대 결론 — VELO 케이스

시계열우선 변수평가결론
1개월내러티브Raptor 4 + 5/20 어닝 + SpaceX IPO진입
3개월내러티브 + Float케인즈 모드, SIDU/SATL 패턴진입
6개월균형PIPE 위험 + 내러티브 소진균형 보류
12개월펀더GP -16%, 추가 PIPE 가능회피

같은 종목인데 시계열만 바꾸면 진입 ↔ 회피가 뒤바뀜. 시계열 결정이 의사결정의 출발점이라는 증거.

흔한 함정

  1. 시계열 무의식 결정 → 분석 일관성 깨짐. 펀더 변수와 내러티브 변수 혼재
  2. 시계열 도중 변경 → 단기 진입했다가 폭등하면 코어 승격 욕망. 시계열이 바뀌면 분석 잣대도 바뀌어야 함을 망각
  3. 펀더로 들어갔는데 단기 하락 못 견딤 → 시계열 어긋남. 처음부터 단기였던 것
  4. 내러티브로 들어갔는데 펀더 미스 우려 → 시계열에 맞지 않는 과도한 펀더 분석
  5. 시계열 미정의 분석 요청 → AI(Bagger)에 묻기 전 본인 시계열부터 명확히

적용 체크리스트 (진입 전)

  • 이 베팅의 시계열을 한 단어로 명시할 수 있나? (1주 / 1개월 / 3개월 / 6개월 / 12개월+)
  • 시계열에 맞는 우선 변수를 식별했나?
  • 시계열을 도중 바꿀 트리거를 사전 정의했나? (예: “6주 안에 +50% 도달 시 시계열 연장 검토”)
  • 시계열 종료 조건(타임스탑)을 entry-memo-3lines에 박았나?

시계열 → 카테고리 매핑

invest-principles-v2-4 카테고리 한도와의 연결:

  • 단기스윙 슬롯 (5-10%): 시계열 1주~3개월, 내러티브 우선
  • 이벤트 드리븐 (5-10%): 시계열 1-6개월, 내러티브 + Float 우선
  • 복권형 (5%): 시계열 무관 (이진 outcome), 내러티브 우선
  • 코어 (25%): 시계열 6-12개월+, 펀더 + 내러티브 align 필수

메타 시사점 — 사용자가 잘하는 영역

장기투자(펀더 베팅)는 시간이 친구지만, 사용자가 즐기지 않음 → 강점 아님. 단기투자(내러티브 베팅)는 시장 심리 읽기가 친구 → 사용자 ENTJ 분석/체계 + 본질 탐구 + 새 것 발굴 성향과 align. 즉 본인 스타일에 맞는 영역은 시장 심리와 매핑 추적이지 펀더 디테일이 아님. 시간 배분도 이 비율로 가야 함.
