ESS Power-for-AI 변동성 프레임

MS “Power for AI” 시리즈 6번째 주자(=ESS) 리포트(2026-03)에서 추출한 정적 프레임. 케이스 디테일은 2026-04-25-dons-ess-6th-reflection.

프레임 갱신 — 절대수요 → 변동성

MS 자체 인용: “prior work focused on identifying the scale of data center power demand. The new insight is that AI inference workloads are now inflecting, structurally increasing power volatility rather than just absolute demand.”

→ “얼마나 많이”가 아니라 “얼마나 유연하게” 가 새 프레임. 셀사이드가 자기 프레임을 명시적으로 갈아엎는 것 자체가 구조 변화 시그널.

트레이닝 vs 추론 부하 패턴

차원트레이닝추론
시간 민감성며칠 멈춰도 OK0.3초 내 응답
스케줄링가능 (밤·저렴 지역)불가
부하 곡선평탄고베이스라인 + 톱니파 + 분 단위 수직 급상승 + 고원 진동
그리드 적합기존 설계 OK기존 그리드 설계 무시

→ 반응 속도 위계: 원전(시간+) > 가스터빈(분) > ESS(밀리초). 추론 부하는 ESS 없이 못 받는다.

시간 우위 (Time-to-Power)

자산배치 시간
가스터빈5년 대기, 가격 2배 (2028 캐파 완판, $3,000/kW)
원전10~15년
그리드 연결5년+
ESS6~12개월 모듈형

→ 미국 2025-28 부족분 918GW (배치량의 1225%가 불 안 켜짐). ESS의 속도 우위가 절대적.

Building Inventory vs Building Capacity (상품 논리 이식)

원자재·상품업계 원칙: 수요가 들쭉날쭉할 땐 설비가 아닌 재고를 쌓는다.

Capacity (설비)Inventory (재고)
되돌릴 수 없음유연
고정비변동비
가동률 리스크수요 스윙 대응
좌초자산 위험가격 규율 유지
자본 투입 큼자본 투입 작음

→ ESS = “전기의 재고”. 발전 대체가 아니라 시스템 균형추 이동.

파생: 재고 논리가 설비 논리를 이기면 경쟁력이 셀 품질·사이클 수명·열화·소프트웨어 최적화로 이동. 단순 GWh 캐파보다 중요해지는 순간이 온다.

Infrastructure Deferral Value (CAPEX 연기 옵션)

ESS 투자는 단순 원가가 아니라 CAPEX 연기 옵션.

  • $10M 투자 5년 지연 → 순 지연가치 $1.15M (인플레 회피 $684K + 이자절감 $41K + 시간가치 $425K)
  • 원 투자금의 7.5~12% 옵션 가치
  • 미국 3대 ISO 송전 증설 $75B → ESS 지연가치 $5.6~9B

→ ESS 단독 LCOE가 가스 피커보다 약간 비싸도 시스템 전체 비용으로는 ESS가 낫다. CFO 관점.

$/kWh → $/cycle 가치평가 전환

소듐이온이 SOC 전구간 출력 일관성을 주면 $/cycle (사이클 경제성) 이 새 척도. 단순 $/kWh 비교는 한계.

  • 소듐 변곡점 202628: Rmb 0.5 → 0.3 이하 (2530% 추가 하락)
  • LCOE 추가 -30%
  • 가스 피커뿐 아니라 CCGT 기저부하까지 잡아먹는 경쟁력

출하 vs 설치 갭 (A/B 시나리오)

2025 출하 612GWh vs 설치 325GWh (갭 287GWh, 88%).

가능성의미결과
A — 중간 재고 적체셀 → 통합 → 현장 이동 중설치량이 출하량 따라잡고 리튬 수요 폭발 → 리튬 $28.5K/t
B — 중국 셀 선제 양산OBBB 전 미국 파이프라인 밀어넣기 + 중국 저가 시장범용 ESS 셀가 하방 압력

→ 베팅 결정 시 A/B 비중 추정 + 리튬 가격 추적이 우선순위.

베타 위계 (베팅 각도)

레이어베타특성
리튬 상류 (간펑·톈치·자금·화우)가장 높음불 케이스 가격 먼저 반영
셀 메이커 (CATL·LGES·BYD·삼성SDI)중볼륨 ↑ 확실, 마진 불확실

가스터빈 출구 타이밍 (페어 트레이드)

  • GE Vernova·SiemensEnergy·MHI = 지난 2년 Power-for-AI 최대 수혜
  • ESS가 가스 피커 수요 상단을 먹기 시작 → 신규 수주 강도 약화 가능
  • 백로그(GE Vernova 100GW)는 3~5년 실적 확실
  • → 밸류에이션 꽉 차면 로테이션 타이밍. ESS 수주잔고 분기 추적이 트리거.

비판적 체크 — 함정 3개

  1. 추론 전력 예측 불확실성: DeepSeek 류 효율 혁신으로 추론 전력이 반토막 나면 ESS 수요도 반토막
  2. ESS 마진 구조적 제약: Fluence 사례 — 배치량 CAGR 33%인데 그로스 마진 10~15%. 볼륨 ≠ 수익
  3. 정책 리스크: 미국 ITC 30% 무너지면 IRR 한자리. 중국 용량요금제 전국 확대 지연 시 IRR 흔들림

사용 원칙

  • ESS 직접 베팅 시 A/B 시나리오 비중 + 정책 리스크 + 마진 구조 3축 사전 점검
  • picks-and-shovels-3layer 3층 프레임에서 ESS는 2차 (설비·소재·컴포넌트), 리튬은 1차 상류 (supply-microstructure-checklist)
  • 한국 베팅 컨센서스: LGES 베팅 / 삼성SDI 관망 / SK이노 회피 (MS 기준)
  • 매크로 환경에서 “빨리 배치 + 연료비 무관 + 자본 효율” 자산 = 지정학·고금리·AI capex 지속 환경에서 프리미엄
