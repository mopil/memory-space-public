Keynesian Beauty Contest Frame (Narrative-Only Trading)

핵심 명제

narrative-only 펌핑 종목은 펀더 박살난 경우가 대부분 (적자, 소형, 희석 이력). 평가 시 “내가 예쁘다고 생각하는 종목”이 아니라 “다수가 예쁘다고 생각할 종목”을 골라야 함. 케인즈 미인 콘테스트.

케인즈 원문 (1936, General Theory): 신문 미인 콘테스트에서 상금을 받으려면 자기 취향이 아니라 다른 응답자들이 가장 예쁘다고 생각할 사람을 골라야 한다. 더 영리한 사람은 “다른 응답자들이 다른 응답자들의 추측을 어떻게 추측할지”를 푼다.

적용 조건

  • narrative 모멘텀 트레이딩 (펀더 베팅 X)
  • 회전율 높은 단기 스윙 (3-6개월 이내)
  • 시드 5-10% 이내 비중 (절대 코어 X)

펀더 평가 방식 전환

일반 종목 펀더: 매출성장, 마진, FCF, ROE, 부채비율 풀스택

narrative-only 종목 펀더 = 오퍼링/희석 리스크 단일 변수

평가 항목:

  • Cash runway (분기 burn rate × 4분기 = 1년치 있나)
  • 최근 PIPE/오퍼링 이력 (12개월 내 빈도)
  • 대주주 락업 만료 시점
  • ATM facility 활성 여부
  • 부채 만기 vs 현금 보유

왜 오퍼링만 보면 되는가

  • 적자 자체는 narrative 펌핑 막지 않음 (SIDU GP -168%도 +854% 폭등)
  • 적자는 시장이 이미 가격에 받아들인 사실
  • 그러나 오퍼링/PIPE는 narrative 형성 중 발사되면 즉사 트리거
  • 희석 발표 = 수급 직접 파괴 = narrative 무관하게 -30~50%

실증 데이터 (2025-2026 우주 narrative)

SIDU: 매출 3.4M / GP -168% / EV/Rev 50.8x / 직원 99 → $0.628 → $5.99 (+854%) SATL: 매출 17.7M / GP +72% / EV/Rev 47.9x / 직원 154 → $1.255 → $8.35 (+566%) VELO: 매출 46M / GP -16% / EV/Rev 7.4x / 직원 134 → $2.81 → $23.84 (+748%)

공통: 100-150명 소형, 적자 수년 누적, Float 작음 (대주주 집중), 단일 narrative 핵심.

펀더 좋고 나쁨이 펌핑 폭과 연관 없음. SIDU(GP-168%)가 SATL(GP+72%)보다 더 큰 배수 폭등.

차트 패턴 (SIDU/SATL 공통)

1차 펌프 (+200500%) → 50% 조정 → 2차 펌프 (1차 고점 돌파 +3080%)

1차 펌프 놓쳐도 50% 조정 시점에 진입 가능. 2차 펌프가 EV 가장 좋음.

평가 체크 5개

  1. narrative 핵심을 한 문장으로 표현 가능한가? (군중 흡수성)
  2. 시장 참여자 픽업 단계는? (디시 → 자동봇 → 매크로채널 → 매체)
  3. 오퍼링 발사 가능성 시점은? (단일 실패 점)
  4. Float 얼마나 작은가? (squeeze 폭 결정)
  5. 1차 펌프 후 50% 조정 패턴 보이나? (2차 기회 진입)

사이즈 가이드

  • 단기스윙 슬롯 5-10% 한도 (절대 초과 X)
  • -30% 손실 가능성 상시 가정. -50%까지도 가능
  • 익절 다단: 1차 +3040% (1/3 트림), 2차 +70100% (1/2 트림), 3차 +150%+ 잔여
  • 무효화 시그널 = 추가 PIPE 발표 → 즉시 절반 이상 청산
  • 타임스탑 4-8주 권장

흔한 함정

  • 펀더 보수주의: “GP -16%니까 위험” → narrative 종목엔 부적절
  • 자기 취향 베팅: “내가 봐도 멋있다” → 군중 픽업 안 되면 횡보
  • 코어 승격: 단기스윙이 폭등하니 비중 늘려 코어로 → 다음 PIPE에서 박살
  • narrative 겹침 무시: 같은 테마 종목 2개 동시 보유 = 분산 효과 없음

사용자 적용 케이스 (2026-04-30 발견)

VELO 분석 시 1차 답변에서 펀더 보수주의로 “하방 닫힘 X” 결론. 사용자가 SIDU/SATL 케이스 반박: “이런 개잡주는 펀더가 다 씹창나있음. 오퍼링 리스크만 중점적으로 보면 narrative 트레이딩 가능.”

데이터 검증: VELO는 SIDU/SATL 대비 EV/Rev 6.5x 저평가, narrative 강도는 상위 (Raptor 4 직접 supplier). 단기스윙 슬롯 8-10% 진입 적합.
