신용 창조 + 카운터파티 프레임 — WOLF 케이스 도출

문제 정의

표면 시그널: short interest 11M > tradeable float 7M = “GME급 squeeze 임박” 실제: borrow rate 1.79% (IBKR), available 1.2M주 = 정상 상태, squeeze 압력 거의 없음

표면 숫자만 보면 진단을 정반대로 내릴 수 있는 함정 구조. 이 함정을 5단계로 해부한 결과를 정리.

5단계 분석 시퀀스

1단계: 표면 시그널 의문 제기

“공매도 수가 거래 가능 주식 수보다 많은 게 어떻게 가능한가?” → 단순 숫자 비교는 결론 도출에 불충분. 메커니즘 분해 필요.

2단계: 신용 창조 메커니즘 (rehypothecation)

같은 주식 1주가 lending 풀에서 여러 번 빌려짐.

실물: 1주
↓
A 빌려줌 → 공매도자 매도 → 매수자 1
↓ 매수자 1의 broker가 또 빌려줌
공매도자 2 매도 → 매수자 2
↓ 무한 반복

시장에 보이는 (+) 권리: 3개+ 시장에 보이는 (-) 의무: 2개+ 실물 1주는 변하지 않음

은행 예금-대출 승수와 동일 메커니즘. MBS, repo, DeFi도 같은 원리.

3단계: 자산 보존 법칙

(+) 권리만 보면 자산이 부풀어 보이지만, (-) 의무 합치면 실물과 일치.

주체자산 (+)부채 (-)순자산
원 소유자1주0+1
공매도자01주 갚을 의무-1
매수자1주0+1
합계211

시장 가격에 반영되는 건 (+)만 카운트하는 경향. 그래서 부풀어 보임.

거품 = (+)와 (-)가 분리되어 인식. 폭락 = 동시 인식되는 순간.

4단계: Borrow rate가 진짜 squeeze 시그널

표면 short ratio (%float) ≠ 실제 squeeze 압력.

진짜 시그널:

  • borrow rate 폭등 (50%+, 정상은 1~5%)
  • available shares 급감 (수천 주 이하)
  • recall 빈도 증가
  • DRS/cold storage 인출 증가

WOLF 비교:

  • ShortInterestTracker (5개월 전 stale): 932% / 1K shares
  • ChartExchange (IBKR 실시간): 1.79% / 1.2M shares
  • 폭등 시점에도 borrow rate 안 올랐음 = 진짜 squeeze 아님

GME 2021은 borrow rate 80%+ + DRS 인출 폭증. WOLF는 둘 다 없음.

5단계: PE 대주주 인센티브 (챕터 11 emergence 종목)

WOLF의 Apollo = 게임 운영자 위치.

Apollo의 다층 수익 구조:

  1. lending fee (연 1.79% × 보유 주식 = 연 100억원급 수익)
  2. 가격 변동성 (공매도 활발 → 변동성 → 단기 차익 기회)
  3. recall right (스퀴즈 버튼 보유, 누르지는 않지만 보험)
  4. 다이루션 wall 풀릴 때 매도 (cost basis $12 → 현재 $48 = 4배 차익)

Apollo가 lending 풀 적극 공급 = 공매도 세력에게 우호적:

  • borrow rate 낮게 유지 → 공매도 존버 가능
  • 다이루션 wall 풀릴 때까지 시간 충분
  • 가격이 단기 폭등해도 Apollo는 더 좋은 가격에 매도

시장 매수자(retail) = 가장 약한 카운터파티. Apollo + Prime Broker + 공매도 세력의 게임에서 정보 비대칭 최하위.

메타 원칙

”정교하고 차갑다”

  • 시스템에 선악 없음. 각자 인센티브에 충실할 뿐.
  • 모두가 합리적, 그러나 약자에게 손실
  • 자비심 기대 = 실망. 인센티브 추적 = 무기

카운터파티 의식

진입 전 항상 질문:

  1. 내가 사면 누가 파나?
  2. 그 사람은 왜 지금 파나?
  3. 그 사람의 cost basis는?
  4. 그 사람의 매도 trigger는?
  5. 정보 비대칭 어디?

특히 챕터 11 emergence 종목:

  • PE 펀드(Apollo, Cerberus, Oaktree 등)가 신주 95% 받음
  • 이들은 “장기 가치 창출”이 목적이 아님 → “차익 실현”이 목적
  • retail이 들어가면 카운터파티가 PE펀드

적용 체크리스트 (다른 종목 분석 시)

챕터 11 emergence 종목 발견 시

  • 신주 발행 비율 확인 (구주주 wipe 정도)
  • 신주 95% 받은 채권자 누구? (Apollo? Oaktree? Cerberus?)
  • PE펀드 cost basis 추정 (전환가/행사가)
  • Registration Rights Agreement 락업 일정
  • 다이루션 wall 크기 (전환사채 + 워런트 + LTIP)

숏스퀴즈 시그널 검증 시

  • Borrow rate (ChartExchange/IBKR 실시간)
  • Available shares 급감 여부
  • 데이터 소스 시점 (stale vs 실시간 구분 필수)
  • DRS 등록 트렌드
  • Recall 활동

카운터파티 매핑

  • 시장 매수자 누구? (retail/기관/PE)
  • 매도 가능 매물 누구 손에?
  • 정보 우위 누구한테?
  • 내 진입가 vs 카운터파티 cost basis 차이

사례별 패턴

Type A: PE 대주주 게임 (WOLF, Hertz post-BK 등)

  • 표면 squeeze 시그널 가짜 가능성 높음
  • Lending 풀 공급 풍부
  • 다이루션 wall이 squeeze 무력화
  • 진입 패스 권장

Type B: 진짜 squeeze (GME 2021, AMC 등)

  • Retail이 lending 풀에서 주식 인출
  • DRS 등록 폭증
  • Borrow rate 80%+
  • 단기 진입 검토 가능 (사이즈 작게, 타임스탑 짧게)

Type C: 정상 가격 발견 (대부분 종목)

  • Lending 풀 정상 작동
  • 시스템 균형 상태
  • 평가는 펀더멘탈 + 모멘텀 기준
