Investing Theses — Meta Index

theses/ 폴더의 투자 아이디어를 축별로 분류한 항해 지도. theses는 시장·종목을 **“어떻게 읽나”**의 해석 관점 일체 — 시장 해석 프레임(렌즈) + 매크로·섹터 관점 + 발굴 도구 + 특정 종목/테마 논거.

세 폴더의 경계: 판단 정체성은 principles-meta-index, 집행 규약은 action-rules-meta-index, 시장 관점은 여기. 이론 토대는 foundations-meta-index.

호출 순서: 사이클 위치 확인 → 매크로 진단 → 수급·내러티브 확인 → 섹터·종목 발굴 → 패턴 인식 → 종목 논거 확정.

1. 사이클 위치 (Cycle Position)

지금 어디인지 — 시간축 위 좌표.

  • eric-peters-2cycle — One River 2사이클. 1사이클 ~2026 9/10월, 2사이클 2027.
  • ai-supercycle-15y-4stage-roadmap — AI 슈퍼사이클 15년 4단계 (confidence 0.4).
  • ai-capex-fcf-chicken-game — 빅테크 FCF (+)→(-) 전환. 사이클 후반 강화 신호.
  • market-regime-framework — Bull/Neutral/Bear 매트릭스 + 진입·이탈 시그널.
  • market-regime-strategy-catalog — 4국면 × 25전략 카탈로그 (해주갤).

2. 매크로·이벤트 (Macro & Cross-Asset)

외부 환경 — 거시 변수와 충격.

  • stagflation-supply-shock-timelag — 스테그 공급충격 + 종전→인플레 6개월 타임랙.
  • yen-carry-unwind-mechanism — 엔캐리 506조엔 + 2024-08 BTC -15% 사례.
  • credit-creation-counterparty-frame — 신용창조 rehypothecation + 카운터파티 5단계 (WOLF 케이스).
  • geopolitical-shock-rally-pattern — 지정학 충격→선반영→해소 랠리 + 22년 반례.
  • market-ignores-macro-until-threshold — 시장 서사 vs 누적 수치 2축. 임계점까지 무시.
  • macro-scenario-matrix — 바이너리 이벤트 전 A/B/C 시나리오 + 비중 사전 배정 → 기계적 집행.

3. 수급·미시구조 (Flow & Microstructure)

누가 사고 파는지 — 가격 이전의 진실.

  • institutional-flow-3metrics — 거래대금/거래량/시총비중 3지표 + 시간축별 우선순위.
  • foreign-rebalancing-cascade-frame — 외인 비중조절 + 매도주체 4분류 + 바스켓 cascade 3 layer.
  • pension-flow-discovery-workflow — KRX 연기금 매매 → 종목 발굴 8단계.
  • ipo-liquidity-drain-pattern — 대형 IPO 유동성 흡수. 리비안 원형, SpaceX 적용.

4. 내러티브·심리 (Narrative & Sentiment)

사람들이 뭘 믿는지 — 가격을 만드는 이야기.

  • narrative-magnet-pattern — 5단계 (씨앗→확산→first mover→매핑→수확). 삼성SDI 4.4배 케이스.
  • keynesian-beauty-contest-narrative — narrative-only 트레이딩 프레임. SIDU/SATL/VELO 실증.
  • dfl-dual-market-narrative — 현물-선물 이중 서사. 시간축 베팅 공존.
  • logic-vs-narrative-divergence — 섹터별 논리 vs 내러티브 align/분리 진단 + 사이즈 가이드.

5. 발굴·섹터 (Discovery & Sector)

종목 찾는 도구 — 어디를 팔지.

  • picks-and-shovels-3layer — 삽과 곡괭이 3단계 벨류체인. 2026-04 ESS 6번째 주자 매핑.
  • sector-leader-identification — 섹터 대장 식별 3축(거래량/눌림/반등) + LITE vs COHR.
  • optic-2stage-frame — 광학 2단계. 1단계(모듈) vs 2단계(컴포넌트).
  • ess-battery-value-chain — ESS/2차전지 4소재 + 셀 + 한국 위상.
  • ess-power-volatility-frame — Power-for-AI Inventory vs Capacity + 출하vs설치 A/B.
  • ai-infra-bottleneck-2030-frame — AI 인프라 병목 2030. L0~L6 + 5루프 + 병목 이동 명제.
  • sector-cycle-saturation — “한 번도 안 해먹은 섹터” 우선. 재진입 동력 약화 진단.

6. 패턴 (Pattern)

자주 반복되는 차트·가격 패턴.

  • parabolic-last-leg-pattern — 10배+ 마지막 레그 +50%. 2/3 홀드 + 1/3 트림.

7. 해석 원리 (Interpretation Principle)

하방·상방을 무엇으로 정의하는가의 메타 프레임.

  • floor-ceiling-time-axis — 펀더=하방, 내러티브=상방. 시계열이 두 변수 영향력 결정. 의사결정 출발점.

8. 종목·테마 논거 (Stock/Theme Theses)

특정 종목/테마를 왜 사는가 — 구체 투자 논거.

  • tenbagger-optic-thesis — AI 광학 멀티배거. Memory Wall → CPO 필수화.
  • aehr-wlbi-thesis — AEHR WLBI 독점 + CoWoS/HBM/photonics 변곡점 + 진입 트리거.
  • ai-liquid-cooling-2phase-thesis — AI 액냉 1-phase vs 2-phase. 2027 GTC catalyst.
  • obbb-feoc-lfp-alpha — OBBB FEOC × 한국 LFP 양극재 알파. 엘앤에프 1차 직격.

action-rules ↔ theses 다리

theses는 해석 관점, action-rules는 행동 룰. 같이 호출:

  • picks-and-shovels-3layer + sector-cycle-saturation — 안 해먹은 섹터를 3 layer로 분해.
  • narrative-magnet-pattern + narrative-ir-alignment-check — narrative 전에 IR 정렬.
  • parabolic-last-leg-pattern + burger-time-sell-rule — 마지막 레그 트림은 미장 본장에서.
  • institutional-flow-3metrics + category-mapping-validation — 수급으로 발견 → 4 게이트 검증.
  • 종목 논거 확정 후 → thesis-4axis-validation-frame로 어닝 검증 룰 세팅.
